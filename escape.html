UX/UI DESIGN

Fitting UI to Art Aesthetic

Escape utilizes an entirely red color palette, so I created the UI art to match that theme. Since the color palettes of the background and the UI elements are the same, the UI practically fades into the background when the player is focused on something else.
When dealing with interactive objects within the game world, I broadened the color palette slightly to give these elements a brighter and more noticeable shade of red. This allows the important elements within the scene to be seen more easily.
Taking UI a Step Further

It is also possible for the UI to come forward and add to the game experience, rather than simply stay out of the way. Doing this requires imaginative solutions that depend entirely on the situation. For example, in Escape there is a moment when the player is forced to kill a rat in order to collect a key it just swallowed. I wanted this moment to be extremely powerful, as it would act as the climax within the narrative. In order to do this, I created a version of the dialogue box that was covered in blood and displayed it as soon as the rat had been killed. This added to the visceral emotion of the moment and left players with their mouths agape.
